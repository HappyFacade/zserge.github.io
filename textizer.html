textizer

Textizer is an app for creating Android widgets.

Basically, it allows you to display any information you want in any style you want and perform any action you want when you tap on the widget.

target audience

Geeks, unix users, scheme fans and other nice guys.

The things you'd better know before starting using textizer:

  • how to edit text files on you phone/table OR
  • how to copy file from your Android device to the PC and back
  • how to write in Scheme (well, it's really easy, so don't worry if you don't know how)

installation

At the moment, textizer is not available on Google Play store. So, please download an *.apk from the Bitbucket page.

Alternatively, you can clone and build the sources:

hg clone https://bitbucket.org/zserge/textizer
cd textizer
android update project -p . -t android-16
ant debug

This will produce an APK in the bin folder.

To create a new widget just add one as you normally do (long tap on the home screen, then pick textizer widget from the list).

Now the tricky part comes.

widget editor

When you create a widget a window will pop up asking for a name for your new widget. Please, use lowercase latin letters and digits. This is the name of the source file corresponding to the widget.

The file itself will be created at <sdcard>/Android/data/trikita.textizer/data/<widget-name.scm>

By default, it's content looks like:

" TODO
()

You are free to customize it any way you like, and I'm trying to provide you with useful Scheme functions.

scheme

Textizer uses jscheme to interpret your widget scripts, which is almost compatible R4RS Scheme language.
